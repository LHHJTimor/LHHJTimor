人固有一死，
或重于泰山，
或轻于鸿毛，
用之所趋异也。
太上不辱先，
其次不辱身，
其次不辱理色，
其次不辱辞令，
其次诎体受辱；
其次易服受辱，
其次关木索，
被菙楚受辱，
其次剔毛发，
婴金铁受辱，
其次毁肌肤，
断肢体受辱，
最下腐刑极矣。
——司马迁《史记》
